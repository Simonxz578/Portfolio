# Portfolio source-material boundary

This directory is reserved for private editorial source material and approved project evidence. It is not part of Astro’s deployed public directory.

Do not commit résumé source files, interview material, participant data, internal company documents, screenshots containing private systems, or unapproved drafts. Publish only verified, public-safe summaries through src/data/en/.

The current portfolio was prepared from a bilingual résumé and a supplemental leadership note supplied outside the repository, the public Cambridge CEO website, public Cambridge Students’ Union directory listings, and the separate Formal Hall Networking Society repository. The private source documents have deliberately not been copied here.
